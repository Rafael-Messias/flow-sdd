# Checklist de Exploracao

Use esta referencia para nao cair em exploracao superficial.

## Cobertura minima

- problema ou decisao principal definida em uma frase;
- contexto atual do repo ou do sistema lido;
- restricoes identificadas;
- riscos e desconhecidos explicitados;
- pelo menos 2 caminhos comparados quando houver mais de uma abordagem plausivel;
- recomendacao final com proximo passo.

## Sinais de boa exploracao

- separa fato de hipotese;
- evita criar artefatos formais cedo demais;
- deixa claro o que ainda precisa ser decidido;
- reduz ambiguidade para o proximo skill do fluxo.
